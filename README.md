# Solar Web Public

Custom integration for Home Assistant to monitor Fronius Solar.web shared/public plant URLs.

## Features

- No Solar.web login required
- No username or password stored in Home Assistant
- One shared URL = one monitored plant
- Main plant entity with attributes
- Separate sensor entities for power, daily energy, total energy and other values
